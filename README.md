# Cognetix_Food-Delivery-Web-app
Foodie Express is a responsive food ordering web application built with HTML, CSS, and JavaScript. It features dynamic menu filtering, real-time cart management, coupon discounts, order tracking, dark mode, food detail modals, and local storage-based order history for an engaging user experience.

# 🍔 Foodie Express

Foodie Express is a modern and responsive food ordering web application developed using HTML, CSS, and JavaScript. The application provides an interactive food menu, shopping cart functionality, order tracking, coupon discounts, dark mode support, and order history management using Local Storage.

## 🚀 Features

### 🍕 Food Menu
- Display multiple food categories
- Food cards with images, ratings, descriptions, and prices
- Dynamic food count display

### 🔍 Search Functionality
- Search foods by:
  - Name
  - Category
  - Description

### 🏷️ Category Filtering
- Pizza
- Burger
- Pasta
- Drinks
- Dessert

### 🛒 Shopping Cart
- Add items to cart
- Increase/Decrease quantity
- Automatic total calculation
- Persistent cart using Local Storage

### 🎁 Coupon System
- Apply discount coupon
- FOOD10 coupon support
- Real-time discount updates

### 📋 Food Details Modal
- View detailed information
- Food image preview
- Ratings and descriptions
- Direct Add to Cart option

### 📦 Order Management
- Order Summary Modal
- Order Confirmation
- Unique Order ID generation
- Estimated delivery time

### 🚚 Live Order Tracking
- Order Received
- Preparing Food
- Out For Delivery
- Delivered

### 🌙 Dark Mode
- Toggle between Light and Dark themes

### 📜 Order History
- Save order history in Local Storage
- View previous orders anytime

### 📱 Responsive Design
- Mobile Friendly
- Tablet Friendly
- Desktop Optimized
